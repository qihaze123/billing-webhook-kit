## Report a Bug

Open a GitHub issue only with fake fixtures and fake secrets:

https://github.com/qihaze123/billing-webhook-kit/issues/new/choose

Do not post:

- Real webhook signing secrets
- Production API keys
- Customer emails or order identifiers
- Private checkout URLs
- Raw production webhook payloads

Read the security policy before reporting sensitive behavior:

https://github.com/qihaze123/billing-webhook-kit/security/policy
